# Chapter 1

Quest/Android/Vulkan port of Chapter 1 from the OpenXR Tutorial series.

- Original tutorial: [Chapter 1: Introduction](https://openxr-tutorial.com//android/vulkan/1-introduction.html#introduction)

---

## Features Demonstrated

This chapter introduces the foundational concepts of OpenXR and establishes a basic project structure. Key features include:

-   **OpenXR Instance Creation**: The initial step to connect an application to an OpenXR runtime.
-   **System and Instance Properties**: Querying the runtime to get information about the XR system and its capabilities.
-   **Basic Project Setup**: Demonstrates how to configure a Gradle and CMake project to compile a minimal OpenXR application for Android.
-   **Application Lifecycle Integration**: Shows how to integrate OpenXR initialization and shutdown with the Android `NativeActivity` lifecycle.
